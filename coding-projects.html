Caiman: Open Source Tool for Scalable Calcium Imaging Data Analysis

Contributed to a major release of the Flatiron Institute's Caiman open-source software package, a python tookit for large-scale Calcium and Voltage imaging data nalaysis. The major release included a significant rewrite of the software's deep learning-based motion correction, source extraction, and Volpy modules, which were from originally written in Tensorflow and were replaced with Pytorch/Keras impelemntations. Significant changes to training and validation pipelines for Volpy's MCRNN and Ring CNN module were also included, and several CNN models were replaced with more efficient and accurate architectures. Still continuing to contribute to the project, including the further improvment of Masked Ring CNNs for the Volpy module.

Neutorch:

Contributed/ing to the development of Neutorch, a python-based Convolutional Neural Network training and inferene kit for Neuron segmentation and synapse detection using Pytorch. Neutorch is a package used for the Flatiron Institute's large-scale Wasp connectomes project, which is a project designed to reconstruct the Megaphragma Viggiani neural system. Currently working on Neuron boundary detection and fixing membrane leaking in Volume Electron Microscopy Images using Neutorch's IsoRSUnet training and validation pipeline.

Quantum Pattern Recognition Algorithm:

Implementation of a quantum algorithm for pattern detection applied to gravitational wave data. Builds parameterized quantum circuits for signal-to-noise ratio estimation and tests them against real gravitational wave data. Built using Python, Qiskit, and Jupyter notebooks with quantum computing libraries.

Quantum Pattern Recognition Algorithm:

Implementation of knowledge distillation pipelines to compress Masked Autoencoder (MAE) models from base/small to tiny architectures for medical image analysis. Includes distillation training scripts, medical image fine-tuning, and evaluation workflows built in PyTorch. Joint work for CS 6998: Advanced Topics in Deep Learning.

Assemblies Ensemble Project:

Simulation of assembly operations as proposed in Papadimitriou et al.'s "Brain computation by assemblies of neurons" (PNAS, 2020). Implements core operations of the Assembly Calculus — project, associate, and merge — on randomly connected populations of neurons with Hebbian plasticity and inhibition. Includes systematic experiments testing assembly behavior across varying configurations of neural firing patterns and parameters.
